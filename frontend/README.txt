
-- INSTRUCTIONS

This frontend has been generated using the official angular-cli (see: https://github.com/angular/angular-cli ).

To build the project:

> npm install
> npm run build

The project is based on Angular.

Technologies involved:
- Typescript as javascript preprocessor
- Webpack as module loader

Nodejs is not required at build or runtime time; however, it is required for local development.


-- BUILD WITH MAVEN

The frontend build is integrated with the maven build thanks to the maven-frontend-plugin.


-- RUN THE DEV SERVER

npm start


-- EXECUTE THE UNIT TESTS

Single execution:
npm run test-single

Continuouos execution:
npm run test
